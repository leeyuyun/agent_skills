# Data Contract

Use UTF-8 CSV files with headers.

## `internal_quotes.csv`

Required minimum columns:
- `trip_id` (or stable route/date fields)
- `origin`
- `destination`
- `depart_date` (`YYYY-MM-DD`)
- `return_date` (`YYYY-MM-DD`, optional for one-way)
- `cabin`
- `internal_price`

Recommended columns:
- `quote_id`
- `internal_airline`
- `internal_depart_time` (`HH:MM`)
- `internal_return_time` (`HH:MM`)
- `internal_stops`
- `agency`

## `public_fares.csv`

Required minimum columns:
- `origin`
- `destination`
- `depart_date`
- `return_date`
- `cabin`
- `public_price`

Recommended columns:
- `trip_id`
- `offer_id`
- `airline`
- `depart_time`
- `return_time`
- `stops`
- `source`
- `url`
- `collected_at` (ISO datetime)

## `watchlist.csv`

Generated by `scripts/build_watchlist.py`.
Core fields:
- `watch_id`
- `trip_id`
- `origin`
- `destination`
- `depart_date`
- `return_date`
- `cabin`
- `preferred_depart_time`
- `preferred_return_time`
- `max_stops`
- `target_currency`

## Matching Rules

Priority order:
1. Match on `trip_id` when both sides provide it.
2. Else match on `origin + destination + depart_date + return_date + cabin`.

Normalize before matching:
- airport codes uppercase
- cabin lowercase
- trim all text values
